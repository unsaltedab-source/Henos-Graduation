# Graduation Ceremony 2025 — Interactive Invitation

An elegant, high-fidelity interactive graduation party invitation website featuring advanced visual animations and MCP (Model Context Protocol) design integration.

## Features
- Sophisticated Design: Elegant navy and gold theme with responsive layouts.
- GSAP Animations: Immersive scroll effects, star-field parallax, and bell-sway transitions.
- Interactive Invitation: Modal invitation experience with staggered entrance animations.
- RSVP Form: Real-time validation and interactive response states with confetti celebration.
- MCP Enhanced: Configuration for Figma and Canva design systems.

## Deployment

### GitHub Pages (Recommended)
This project is configured for automated deployment via GitHub Actions.
1. Push to the main branch.
2. The site will automatically build and deploy to the gh-pages branch.

Note: As GitHub Pages hosts static content, the RSVP backend features are simulated for demonstration.

### Render.com
The repository includes render.yaml for deploying both the static frontend and the Node.js form-processing worker.
- Web Service: Static build from dist/.
- Worker Service: Node.js backend in server/.

## Local Development
Install dependencies: npm install
Start dev server: npm run dev
Build for production: npm run build

## MCP Integration
The mcp-config.json file includes configurations for:
- Figma MCP: For design system generation and high-fidelity mockups.
- Canva MCP: For celebration graphics and social media assets.
